# 缠论分析工具 (chanlun-analyzer)

基于缠中说禅《教你炒股票》108课的A股技术分析工具。

## 功能目标

- K线包含处理
- 分型（顶分型/底分型）自动识别
- 笔的划分
- 中枢自动计算
- 三类买卖点判断

## 目录结构

```
chanlun-analyzer/
├── README.md
├── docs/          ← 算法规则文档
├── chanlun/       ← 核心算法库
├── tests/         ← 测试用例
└── examples/      ← 使用示例
```

## 协作方式

- **智远**：算法规则设计、文档编写、功能验证
- **智诚**：架构审校、进度协调
- **智行**：代码实现

## 数据源

- A股数据：baostock (免费接口)
